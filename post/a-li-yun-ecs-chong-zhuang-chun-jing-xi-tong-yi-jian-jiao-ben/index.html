阿里云ECS重装纯净系统 一键脚本

2020-03-16

  • 功能

    阿里云ECS会默认自动安装阿里云安骑士等，既是安全防护，也是后台监控。为了获得一个没有安骑士的系统，可使用网上大神的一键脚本重装一个纯净系统。

    本文以默认安装了Debian的阿里云ECS重装Debian 9为例。

  • 一键脚本

    • 先更新一下

      apt-get update

    • 安装所需软件

      apt-get install -y xz-utils openssl gawk file

    • 一键重装Debian

      bash <(wget --no-check-certificate -qO- 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh') -d 9 -v 64 -a

      上述一键脚本中-d 9 -v 64表示64位Debian 9，可根据需求自行修改。

  • 修改root密码

    喝杯咖啡等系统重装好后，登陆root账户：MoeClub.org，然后修改root密码：

    passwd root
