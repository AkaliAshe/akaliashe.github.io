V2Ray WS TLS1.3一键脚本

2020-03-16

  • WebSocket+TLS+Web（Caddy）

    从外面看，该方案的流量完全属于通过443端口访问正常的HTTPS网站，TLS可以加密流量，使其看起来更像HTTPS。该组合方案算是目前比较稳定的出国方案了。

  • 方案说明

    • VPS系统要求为Debian 9+/Ubuntu 16.04+。
    • 需要一个域名并解析到VPS的IP。
    • 建议搭配BBR加速。
    • 一键脚本会配置一个英文模版的网站，访问地址即为解析到VPS的域名。

  • 开始吧

    • 更新内核并开启BBR

      wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh

      按提示重启VPS。（若内核较新则不会更新内核，会直接开启BBR，无需重启）

    • 安装curl（若系统已安装curl则无需此步骤）

      apt-get install curl

    • 安装V2Ray+WS+TLS

      curl -O https://raw.githubusercontent.com/atrandys/v2ray-ws-tls/master/v2ray_ws_tls1.3.sh && chmod +x v2ray_ws_tls1.3.sh && ./v2ray_ws_tls1.3.sh

      脚本执行过程中会提示需要输入域名，输入解析到本VPS的域名即可。安装完成后，可以看到配置参数，客户端配置时要用到相关参数。

  • 客户端配置

    • 推荐使用如下客户端

    ​ Windows：Clash for Windows

    ​ Mac：ClashX

    ​ Android：Clash for Android

    ​ iOS：Shadowrocket、QuanrumltX等

    ​ OpenWRT：OpenClash

    • 将配置参数对应填写到客户端
    • 配置所需要的规则
