观看Netflix的Clash规则

2020-03-16

  • 说明

    该规则用于观看Netflix港区内容，仅代理Netflix流量。服务器为阿里云轻量应用服务器-HK，搭建了V2Ray（WebSocket+TLS+Web），搭建方法见V2Ray WS TLS1.3一键脚本。

  • Clash规则（搜集于网络）

port: 7890
socks-port: 7891
allow-lan: false
Proxy:
# vmess
- name: "Ali"
  type: vmess
  server: ****
  port: 443
  uuid: ****
  alterId: 64
  cipher: auto
  # udp: true
  tls: true
  # skip-cert-verify: true
  network: ws
  ws-path: ****
  # ws-headers:
  Host: ****
Proxy Group:
- name: NETFLIX
  type: select
  proxies:
    - Ali
    - DIRECT
